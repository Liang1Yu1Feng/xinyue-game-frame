# xinyue-game-frame

心悦游戏框架
------------------------------------------------------------------------------------------------------------------------

此游戏框架服务器是基于SpringCloud组件开发,使用netty提供网关的功能。

此框框架服务端实现的功能有：
1，注册登陆
2，基于Web服务的网关
3，基于Netty的长连接网关
4，服务端的动态扩展。

客户端使用的是Unity3d.
